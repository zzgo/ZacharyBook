Java各类型占用字节数

| 数据类型 | 二进制位数 | 字节大小 | 范围 | 默认值 |
| :--- | :--- | :--- | :--- | :--- |
| byte | 8 | 1 | -128~127 | 0 |
| short | 16 | 2 |  | 0 |
| int | 32 | 4 |  | 0 |
| long | 64 | 8 |  | 0 |
| float | 32 | 4 |  | 0.0f |
| double | 64 | 8 |  | 0.0d |
| char | 16 | 2 |  | '\u0000' |
| boolean | 1 | 1\(1/8,按1计算\) | true/false | false |
